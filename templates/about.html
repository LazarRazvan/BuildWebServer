This is a tool that allows building large C projects and inspecting statistics of previous builds.


We support two different compilers :

  • GCC
  • CLANG


Each compiler will run in a separate Docker container and will be used to meet the users needs.


We collect build information from each user that are displayed using Grafana. Because of this, each user will be able to inspect the builds history, reviewing errors and warnings.
